¿HTML5?

Es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…


HTML consiste en una serie de códigos cortos, clasificados en un archivo de texto por el sitio del autor – estos son las etiquetas. El texto es guardado como un archivo html, el cual se puede visualizar a través de un buscador, como Internet Explorer o Netscape Navigator.


Sus objetivos principales son mejorar el lenguaje dando soporte para los últimos objetos multimedia mientras se mantiene fácilmente legible por los humanos y a su vez ser comprendido constantemente por ordenadores y dispositivos (navegadores web, programas de análisis, etc.).
